# 🎮 2048 Puzzle Game (Python)

A modern implementation of the classic **2048 Puzzle Game** developed using **Python** and **Tkinter**. The objective is to combine numbered tiles to reach the **2048 tile** while achieving the highest possible score.

---

## 📌 Features

* 🎮 Classic 2048 gameplay
* ⌨️ Keyboard controls (Arrow Keys)
* 🔄 Restart Game button
* 🆕 New Game button
* 🏆 Live Score & Best Score tracking
* 🎉 Win notification
* 💀 Game Over detection
* 🎨 Clean and responsive user interface
* 💻 Lightweight desktop application

---

## 🛠️ Built With

* Python 3
* Tkinter (GUI)
* Object-Oriented Programming (OOP)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/2048.git
```

### Navigate to the project

```bash
cd 2048
```

### Run the game

```bash
python main.py
```

## 🎮 Controls

| Key            | Action       |
| -------------- | ------------ |
| ⬅️ Left Arrow  | Move Left    |
| ➡️ Right Arrow | Move Right   |
| ⬆️ Up Arrow    | Move Up      |
| ⬇️ Down Arrow  | Move Down    |
| R              | Restart Game |
